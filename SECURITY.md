# Security policy

TenantProbe is pre-v1. Security fixes currently target the latest `main` branch.

Please report vulnerabilities using GitHub's private vulnerability-reporting
feature for this repository. Do not open a public issue containing credentials,
private target URLs, or exploit details. If private reporting is unavailable,
open a minimal issue asking the maintainer for a private contact channel.

Only test systems you own or are explicitly authorized to assess. Use synthetic
fixtures and a disposable staging environment whenever possible.
